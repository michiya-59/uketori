# ウケトリ（UKETORI）システム概要資料

> 見積から入金まで、ぜんぶウケトリ。

---

## ウケトリとは？

ウケトリは、中小企業・フリーランスのための**受発注・請求・入金回収管理**クラウドサービスです。

見積書の作成から、発注・納品・請求、そして**入金の確認**まで、日々の取引業務をひとつの画面で完結できます。

特に「請求書を送ったのに入金されない」「誰に何を催促すればいいかわからない」といった**入金回収の悩み**を解決するために設計されています。

---

## こんなお悩みを解決します

| よくある課題 | ウケトリでの解決方法 |
|-------------|-------------------|
| Excelやスプレッドシートで帳票を管理していて、ミスが多い | クラウド上で見積書〜領収書まで一元管理。自動採番・自動計算で入力ミスを削減 |
| 請求書を送った後、入金確認が手作業で大変 | 銀行明細CSVを取り込むだけで、**AIが自動で請求書と入金を照合** |
| 支払い遅延の顧客への催促を忘れてしまう | 支払期限を過ぎたら**自動でリマインドメールを送信**。催促漏れゼロに |
| どの請求書がまだ回収できていないか把握できない | **回収ダッシュボード**で未回収額・回収率・遅延状況をリアルタイムに可視化 |
| 他のツールからの乗り換えが面倒 | **AIが自動でデータを変換**。最短5分で移行完了 |

## ウケトリの主な機能

### 1. 帳票管理（見積書〜領収書）

ビジネスで必要な6種類の帳票をすべて作成・管理できます。

| 帳票の種類 | 説明 |
|-----------|------|
| 見積書 | お客様への金額提示。承認後、そのまま請求書や発注書に変換可能 |
| 発注書 | 仕入先への注文書 |
| 注文請書 | 発注に対する受諾書 |
| 納品書 | 商品・サービスの納品時に発行 |
| 請求書 | 代金の請求。インボイス制度（適格請求書）対応 |
| 領収書 | 入金確認後に発行 |

**帳票のかんたん変換**

帳票はワンクリックで別の帳票に変換できます。同じ内容を何度も入力する必要はありません。

```
見積書 → 発注書 / 請求書
発注書 → 納品書 / 請求書
請求書 → 領収書
```

**インボイス制度対応**

適格請求書発行事業者登録番号（T+13桁）を設定に登録しておけば、発行する帳票に自動で記載されます。国税庁のデータベースとの照合機能もあります。

**PDF出力・メール送信**

作成した帳票はPDFとしてダウンロード、またはそのままメールで取引先に送信できます。

---

### 2. 顧客管理

取引先の情報を一元管理します。

- 会社名・住所・連絡先・担当者情報の登録
- 顧客ごとの取引履歴（見積書〜請求書）の確認
- 顧客タイプの分類（顧客 / 仕入先 / 両方）
- 複数の担当者連絡先を登録可能

---

### 3. 案件管理

取引をプロジェクト（案件）単位で管理します。

- 案件の作成・進捗管理（見積中 → 受注 → 進行中 → 納品 → 請求 → 完了）
- 案件に紐づく帳票の一覧表示
- 担当者のアサイン
- パイプライン（案件の進捗状況を一覧で把握）

---

### 4. 入金管理

請求書を送った後の入金状況を管理します。

- 請求書ごとの入金ステータス表示（未入金 / 一部入金 / 入金済 / 遅延）
- 入金の手動記録（金額・日付・方法）
- 入金額と残額の自動計算

---

### 5. 銀行明細の取込とAI自動消込

ウケトリの代表的な機能のひとつです。

**使い方はかんたん3ステップ：**

```
ステップ 1：銀行のインターネットバンキングからCSVをダウンロード
ステップ 2：ウケトリにCSVをドラッグ＆ドロップ
ステップ 3：AIが自動で「どの請求書への入金か」を判定 → 確認して完了
```

**AIの照合方法：**

1. **金額の一致** — 請求金額と入金金額がぴったり合う場合は自動マッチング
2. **名前の照合** — 振込人名と顧客名を照合（カナ表記の揺れにも対応）
3. **AI判定** — 上記で判定できない場合、AIが総合的に判断
4. **自動/手動の振り分け** — 確信度90%以上は自動消込、それ以下は確認画面で手動確認

銀行のCSVフォーマットは主要銀行に対応しており、文字コード（Shift_JIS / UTF-8）も自動で判別します。

---

### 6. 自動督促（リマインドメール）

支払期限を過ぎた請求書に対して、自動でリマインドメールを送る機能です。

**設定例：**

| 条件 | アクション |
|------|-----------|
| 支払期限を 3日 過ぎたら | お客様にやんわりリマインドメール |
| 支払期限を 14日 過ぎたら | 少し強めの督促メール |
| 支払期限を 30日 過ぎたら | 最終通知メール + 社内アラート |

- メールの文面はテンプレートで自由にカスタマイズ可能
- 顧客名・請求金額・支払期限などは自動で差し込み
- 送信回数の上限や送信間隔も設定できるため、過度な督促を防止
- 督促の送信履歴はすべて記録に残ります

---

### 7. 回収ダッシュボード

お金の回収状況をひと目で把握できるダッシュボードです。

**表示される情報：**

| 指標 | 説明 |
|------|------|
| 未回収合計 | まだ入金されていない請求書の合計金額 |
| 遅延金額 | 支払期限を過ぎた請求書の合計金額 |
| 回収率 | 当月発行した請求書のうち、回収できた割合（%） |
| 平均回収日数 | 請求書発行から入金までの平均日数 |

**売掛金年齢表（エイジングレポート）：**

顧客ごとに「どれくらい支払いが遅れているか」を一覧で表示します。

```
                    当月  1〜30日  31〜60日  61〜90日  90日超
A株式会社           50万    10万     -         -        -
B商事                -      -       30万      20万      -
C工業                -      -        -         -       15万  ← 要注意
```

色分けされているため、危険な取引先をすぐに見つけることができます。

---

### 8. 与信スコア

顧客の支払い信頼性を 0〜100点 で自動スコアリングする機能です。

- 支払い履歴（期日内の入金、遅延回数など）に基づいて自動計算
- スコアが大きく下がった場合は通知でお知らせ
- 回収ダッシュボードの「要注意顧客」一覧と連動

**スコアの目安：**

| スコア | 評価 | 意味 |
|--------|------|------|
| 70〜100 | 良好 | 支払いが安定している優良顧客 |
| 50〜69 | 注意 | たまに遅延がある顧客 |
| 30〜49 | 警戒 | 遅延が目立つ顧客。早めの対応を推奨 |
| 0〜29 | 危険 | 深刻な遅延がある顧客。取引条件の見直しを検討 |

---

### 9. データ移行（他ツールからの乗り換え）

現在お使いのツールやExcelからのデータ移行を、AIの力でかんたんに行えます。

**対応ツール：**
- freee
- board
- Misoca
- MakeLeaps
- Excel / CSV（フォーマット自由）

**移行の流れ（5ステップ）：**

```
ステップ 1：移行元を選択（わからなければ「自動判定」でOK）
ステップ 2：CSVファイルをドラッグ＆ドロップ
ステップ 3：AIが自動でカラムを対応づけ → 確認画面で微調整
ステップ 4：取り込みデータのプレビューを確認
ステップ 5：実行 → 結果レポート表示
```

AIがCSVの列名を読み取り、「この列は会社名」「この列は金額」と自動で判定するため、面倒なマッピング設定は不要です。

---

### 10. ダッシュボード・レポート

ビジネスの全体像を把握するためのダッシュボードとレポート機能です。

- **売上推移**：月ごとの売上金額の推移をグラフで表示
- **入金予定**：今後の入金予定額をカレンダー形式で表示
- **最近の取引**：直近の帳票発行・入金の一覧
- **パイプライン**：案件の進捗ステージごとの金額集計

## 業務フローの全体像

ウケトリを使った日常業務の流れです。

```
┌────────────────────────────────────────────────────────────┐
│                     ウケトリの業務フロー                      │
└────────────────────────────────────────────────────────────┘

  お客様から依頼
       │
       ▼
  ① 見積書を作成 → メール送信 or PDFダウンロード
       │
       ▼ お客様が承認
  ② 見積書 → 請求書に変換（ワンクリック）
       │
       ▼
  ③ 請求書を送付 → メール送信
       │
       ├──── 入金あり ────→ ④ 銀行CSVを取込 → AI自動消込 → 入金完了
       │                         or 手動で入金記録
       │
       └──── 入金なし ────→ ⑤ 自動督促メール送信（設定に基づき自動実行）
                                    │
                                    ▼
                              ⑥ 回収ダッシュボードで状況確認
                                 → 与信スコアの変動をチェック
                                 → 年齢表で遅延顧客を把握
```

## チームでの利用

ウケトリは、チーム内で役割に応じた権限を設定できます。

| 役割 | できること |
|------|-----------|
| オーナー | すべての操作。プランの変更、テナントの管理 |
| 管理者 | ユーザー管理、各種設定の変更 |
| 経理 | すべての帳票・入金・レポートの閲覧と編集 |
| 営業 | 自分が担当する案件と帳票の作成・編集。入金情報は閲覧のみ |
| メンバー | 閲覧のみ |

最大30名までのチームメンバーを招待できます（プランにより上限が異なります）。

## セキュリティ

- **データの分離**：企業ごとにデータが完全に分離されており、他社のデータにアクセスすることはできません
- **暗号化通信**：すべての通信はSSL/TLSで暗号化
- **パスワード保護**：パスワードは業界標準のbcryptでハッシュ化して保存
- **操作ログ**：すべての作成・更新・削除操作を記録。「いつ・誰が・何をしたか」を追跡可能
- **ロールベースアクセス制御**：役割に応じた権限管理で、不必要な情報へのアクセスを制限
